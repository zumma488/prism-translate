# Code of Conduct

## Our Pledge

We want Prism Translate to be a welcoming, respectful, and constructive community for everyone.

Participants are expected to:

- act with empathy and professionalism
- welcome good-faith questions and feedback
- respect different levels of experience
- focus on what is best for the project and community

## Expected Behavior

- Be respectful in discussions, reviews, and issue threads.
- Give feedback about code and ideas without personal attacks.
- Assume good intent, then clarify concerns with specifics.
- Share credit and document decisions when collaboration crosses time zones or languages.

## Unacceptable Behavior

- Harassment, discrimination, or abusive language
- Personal attacks, trolling, or deliberate intimidation
- Publishing private information without permission
- Repeated bad-faith disruption of discussions or reviews

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, open a GitHub issue and redact any sensitive personal or security-related details before posting. If the issue also involves a security concern, follow the sanitization guidance in [SECURITY.md](SECURITY.md).
