# 行为准则

## 我们的承诺

我们希望 Prism Translate 成为一个欢迎、尊重、建设性的协作社区。

所有参与者都应当：

- 以同理心和专业态度参与协作
- 欢迎善意的问题与反馈
- 尊重不同经验水平的贡献者
- 优先考虑项目和社区的共同利益

## 期望行为

- 在讨论、评审和 issue 交流中保持尊重。
- 针对代码和想法提出反馈，不进行人身攻击。
- 先假定对方出于善意，再用具体事实澄清分歧。
- 当协作跨越时区或语言时，及时共享背景并记录决策。

## 不可接受的行为

- 骚扰、歧视或辱骂性语言
- 人身攻击、钓鱼挑衅或故意恐吓
- 未经许可公开他人隐私信息
- 持续以恶意方式破坏讨论或评审秩序

## 执行

项目维护者可以删除、编辑或拒绝违反本行为准则的评论、提交、代码、issue 或其他贡献内容。

## 举报方式

如果你遭遇或目击不可接受的行为，请通过 GitHub issue 反馈，并在发布前去除任何敏感的个人信息或安全细节。如果内容同时涉及安全问题，请遵循 [SECURITY.md](SECURITY.md) 中的脱敏说明。
